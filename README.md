
## Sistema de Compra Simples

Desenvolvido em dupla

---

## Descrição
Este projeto é um **sistema de compras simples em C**. Ele exibe uma lista de produtos, permite que o usuário escolha os produtos pelo código e quantidade, e depois imprime uma nota fiscal com o total da compra.

O objetivo é praticar lógica de programação, estruturas de repetição, arrays e manipulação de strings.

---

## Estrutura do Projeto
- `compra.c` – Código fonte do sistema de compras

---
